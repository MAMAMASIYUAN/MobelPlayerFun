### MobelPlayerFun是一个影音播放器  
<br />该项目为运行于安卓手机系统的影音播放器。该播放器可列出并播放本地视频，本地音频，网络视频，网络音频。并且集成了万能播放器 Vitamio以应对某些系统不支持的视频/音频格式。 
<br />编程语言：Java。 
<br />特色功能:
<br />1. 集成了 xListView以实现下拉刷新，上拉加载更多功能。 
<br />2. 分别尝试了 xUtils3，Glide，Picasso用于获取网络图片。尝试用 EventBus3.0代替 Broadcast. 
<br />3. 使用分类型的 ListView实现类似今日头条的推送内容。 
<br />4. 集成jcvideoplayer 在 ListView中播放视频。 
<br />5. 集成科大讯飞智能语音识别，用于新闻搜索功能。 
<br />6. 集成 AI机器人，实现简单对话。 
<br />开发工具：Android Studio
